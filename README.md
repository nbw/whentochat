# When to chat

![banner](static/banner.png)

A tool to help resolve time zone differences. 

Check it out at [whentochat.co](https://whentochat.co).

## Tech

* Gatsby for static site
* Google Maps for location data
* Hosted on Netlify
* Query Bitly via an AWS Lambda
